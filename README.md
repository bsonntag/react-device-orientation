# @bsonntag/react-lib-boilerplate

> A boilerplate for react modules.

## Installation

Using npm:

```sh
$ npm install --save @bsonntag/react-lib-boilerplate
```

Using yarn:

```sh
$ yarn add @bsonntag/react-lib-boilerplate
```
